# Rule That Cannot Change x Artifact Consumer Matrix

## Purpose

This matrix tracks cybernetic rules that cannot change across the artifacts that must create,
preserve, check, review, compile, or test them.

A rule is not considered maintained when it only appears in prose. For every
new or changed rule that cannot change, update the row that identifies:

- where the rule is stated;
- where the artifact field or template captures it;
- which deterministic guard rejects missing or invalid structure;
- which review dimension judges semantic adequacy;
- which downstream consumer or compiler preserves it;
- which eval or test prevents regression.

For non-applicable consumers, record `N/A: <reason>` and leave no blank cells.
Deterministic guards should check structure and explicit values;
semantic sufficiency belongs to review.
