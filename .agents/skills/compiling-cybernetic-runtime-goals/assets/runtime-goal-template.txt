# JSON Control Guidance: Runtime

This file is not an official control fact template.

Use `runtime.control.example.json` as builder guidance for the compiled runtime
control file under `docs/cybernetics/runs/<slug>/`.

Official run artifacts:

- `requirements.control.json`
- `design.control.json`
- `goal.control.json`
- `plan.control.json`
- `review.control.json`
- `runtime.control.json`
- `progress.jsonl`
- `runtime-status.json`
- `final-report.json`

The runtime entry point is a short `/goal` pointer to `runtime.control.json`
using `.agents/skills/using-control-json`.

Markdown may be used for human-facing explanation, README material, or skill
documentation. It must not be used as guard, compiler, or runtime control input.
